Блочные теги:

  • < html > - основной тег, определяющий начало и окончание документа, заключает в себе все содержимое страницы
  • < head > - тег, в котором содержится техническая информация, помогающая браузеру и поисковым системам. Его содержимое не отображается на веб-странице, за исключением элемента < title >, который дает заголовок окна на веб-странице
  • < body > - тег, предназначенный для хранения содержимого веб-страницы, отображаемого в окне браузера
  • < title > - определяет заголовок документа, не является частью документа и не показывается напрямую на веб-странице. Текст заголовка отображается на вкладке браузера. Допускается использовать только один тег < title > на документ и размещать его внутри тега < head >
  • < header > - задает "шапку" сайта или раздела, в которой обычно располагается заголовок, логотип, адрес, контактная информация.
  • < footer > - заает "подвал" сайта или раздела, в нем может располагаться имя автора, дата документа, контактная и правовая информация.
  • < p > - блочный тег, служит для обозначения текстовых абзацев, абзацы текста идущие друг за другом разделяются между собой отбивкой. Если закрывающего тега нет, считается, что конец абзаца совпадет с началом следующего абзаца или другого блочного элемента.
  • < div > - универсальный блочный элемент, предназначенный для группирования элементов документа с целью изменения вида содержимого через сттили. Для этого добавляется атрибут class или id с именем класса или идентификатора
  • < h1...h6 > - заголовки 1...6 уровня, которые показывают относительную важность секции, расположенной после заголовка. Так, элемент < h1 > представляет собой наиболее важный заголовок первого уровня, а < h6 > служит для обозначения заголовка шестого уровня и является наимене значительным. По умолчанию, заголовок первого уровня отображается самым крупным шрифтом жирного начертания, заголовки последующего уровня по размеру меньше.
  • < ul > - маркированный (не упорядоченный) список
  • < ol > - нумерованный (упорядоченный) список
  • < li > - определяет отдельный пункт списка
  • < pre > - определяет блок предварительно формативного текста. Такой текст отображается обычно моноширинным шрифтом и со всеми пробелами между словами. По умолчанию, любое количество пробелов идущих в коде подряд, на веб-странице показывается как один
  • < nav > - задает навигацию по сайту
  • < section > - Задает раздел документа, может применяться для блока новостей, контактной информации, глав текста, вкладок в диалоговом окне и др. Раздел обычно содержит заголовок. Допускается вкладывать один тег < section > внутрь другого.
  • < aside > - Представляет собой раздел страницы, который имеет косвенное отношение к содержимому страницы и может быть рассмотрен отдельно от этого содержимого. < aside > применяется для боковых понелей, рекламных блоков, ссылок на архив, меток и другой информации, которая отделена от основного содержимого страницы.
  • < article > - задает содержание сайта вроде новости, статьи, записи блога, форума и др.

Строчные теги:

  • < strong > - предназначен для выделения важного текста как стилистически (отображается жирным) так и по содержанию
  • < b > - устанаваливает жирное начертание шрифта
  • < em > - предназначен для акцентирования текста как стилистически (отображается курсивым) так и по содержанию
  • < i > - устанавливает курсивное начертание шрифта
  • < u > - подчеркивание текста. Используется для разметки текста, который должен отличаться стилистически от обычного текста
  • < a > - предназначен для создания ссылок. Для этого необходимо сообщить браузеру, что является ссылкой, а также указать адрес документа, на который следует сделать ссылку. В качестве значения атрибута href используется адрес документа, на который происходит переход. Адрес ссылки может быть абсолютным и относительным. Абстолютные адреса работают везде и всюду независимо от имени сайта или веб-страницы, где прописана ссылка. Относительные ссылки, как следует из их названия, построены относительно текущего документа или корня сайта
  • < span > - предназначен для выделения отдельных строк, символов или других строчных элементов для дальнейшего изменения их оформления с помощью стилей
  • < style > - применяется для определения стилей элементов веб-страницы. Элемент < style > необходимо использовать внутри контейнера < head>
  • < br > - устанавливает перевод стоки в том месте, где он находится. В отличие от элемента < p >, исользование < br > не добавляет пустой отступ перед строкой
